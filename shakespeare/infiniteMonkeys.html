About Infinite Monkeys

An adage:
An infinite number of monkeys banging on an infinite number of typewriters will eventually produce Hamlet.

In that spirit, Infinite Monkeys computationally randomizes Shakespeare's body of work to generate brand new works.

The text comes from Open Source Shakespeare and is processed using Python and JavaScript.

Check out the source code at Github.
